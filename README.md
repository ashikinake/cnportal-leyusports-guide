# cnportal-leyusports-guide

本仓库用于归档和发布多个独立 HTML 页面，涵盖各类资讯、工具与信息展示用途。页面内容以静态 HTML 文件形式存储，可直接通过浏览器访问，无需额外服务器环境。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。所有页面均为独立、自包含的 HTML 文件，不依赖外部数据库或后端服务。仓库本身不指向任何特定域名或网站，仅作为页面集合的集中存储与版本管理之用。

## 目录说明

```
├── README.md          # 本说明文件
├── index.html         # 仓库首页或导航入口（如有）
├── pages/             # 存放主要 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 静态资源（CSS、JS、图片等，可选）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：核心页面目录，每个文件对应一个独立 HTML 页面。
- `assets/`：存放页面所引用的公共资源文件，方便统一管理。

## 页面归档说明

- 所有页面均为静态 HTML 文件，符合 W3C 标准，可脱机浏览。
- 页面之间无强制跳转或依赖关系，每个文件均可独立打开。
- 归档内容以信息展示、工具说明或资料整理为主，不包含任何商业推广、诱导点击或营销性质的内容。
- 如页面中包含外部链接，仅用于引用来源或提供扩展阅读，不构成对第三方网站的背书。
